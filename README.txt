version 1.4(current)
____________________________________________________________________________________
В приложение интегрирован lua, теперь именно lua считывает все данные из файла
конфигрурации. В настройках теперь добавлена возможность устанавливать уровень 
анизотропной фильтрации. Прочие мелкие правки.
____________________________________________________________________________________

version 1.35
____________________________________________________________________________________
Немного изменена интерактивная заставка в главном меню, полностью переработаны все
классы, которые используют класс Sprite, теперь они все не хранят его копии, а 
получают указатель на единственный экземпляр в памяти только во время отрисовки, что
позволило уменьшить использование оперативной памяти примерно на 4-6 мб. Теперь 
память под изображения, текстуры, спрайты и фоны для текста выделяются сразу одним
блоком, получить доступ к которым можно через указатель ptr_global_memory и указав
смещение объекта на кол-во байт от начала блока. Добавлен файл MemoryPool.h для
более удобной работы с памятью. Прочие мелкие улучшения.
____________________________________________________________________________________

version 1.3
____________________________________________________________________________________
В главное меню добавлена интерактивная заставка. Теперь передача класса Color и Font
во все функции осуществляется через константную ссылку. Большинство классов больше
не хранят класс Font внутри себя, а получают его при создании через конструктор.
____________________________________________________________________________________

version 1.28
____________________________________________________________________________________
Добавлен класс Meteor, который является наследником класса ObjectAnimation, в связи
с чем, был перенесен эффект взрыва с ObjectAnimation на Meteor, в меню настроек
(ура! и сто лет не прошло) была добавлена возможность изменять громкость звука и 
возможность отключать и включать его, теперь, при изменении некоторых настроек,
не обязательно перезаходить в игру. Из-за этих нововведенний была доработана 
структура settings и все классы, использующие эту структуру. Проведен рефакторинг
кода и теперь github распознает кириллицу в исходниках. Различные мелкие правки.
____________________________________________________________________________________

version 1.21
____________________________________________________________________________________
Исправлен деструктор класса multiline_text, теперь удаление элементов из вектора 
mass_string производится более корректно.
____________________________________________________________________________________

version 1.20
____________________________________________________________________________________
Теперь все классы, использующие внутри себя класс Texture, будут получать его 
экземпляр через указатель, а не хранить этот экземпляр как часть себя занимая тем 
самым лишнюю память, это в свою очередь позволило уменьшить объём занимаемой памяти 
почти в 2 раза(примерно со 130мб до 55 - 70мб). Была удалена функция SetImage в 
классе Character, за ненадобностью. Переделан метод хранения масивов строк в классе
multiline_text, теперь используется вектор, а не обычный массив Text* как раньше.
Различные мелкие улучшения.
